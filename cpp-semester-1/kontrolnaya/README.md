# Контрольная работа (Первый семестр)

В данной директории представлены варианты и решения задач с промежуточной контрольной работы (или самостоятельной).

## Описание заданий

Контрольная проверяет базовое понимание работы с циклами, функциями и одномерными массивами (передача массивов в функции
по указателю `int *array`).

- **`kr1.cpp` (Точка равновесия)**: Поиск такого индекса в массиве, для которого сумма всех элементов слева равна сумме
  всех элементов справа. Алгоритм показывает оптимизацию со сложностью $O(N)$ вместо двойных циклов за счет
  предварительного подсчета общей суммы.
- **`kr2.cpp` (Числа Фибоначчи в массиве)**: Динамическое заполнение массива числами Фибоначчи и поиск суммы тех
  элементов, чьи значения попадают в заданный диапазон `[from; to]`. Решение проверяется встроенными автотестами.
- **`kr3.cpp` и `kr3Armstrong.cpp` (Числа Армстронга)**: Полноценный алгоритм с функциями возведения в степень, подсчета
  количества цифр и проверки на число Армстронга. Далее эти проверки применяются к массиву для фильтрации и сложения "
  счастливых" элементов.
