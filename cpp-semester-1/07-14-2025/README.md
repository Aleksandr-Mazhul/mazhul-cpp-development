# Практика 07-14-2025 (Вычисление сумм)

В этой папке находится код (`sum.cpp`), посвященный накоплению значений в цикле или вычислению математических сумм/рядов.

## Описание
Задачи типа "Вычислить сумму" (`sum`) обычно подразумевают:
- Итерацию по массиву и сложение определенных элементов (например, положительных или кратных заданному числу).
- Расчет числового ряда по формуле (например, суммы для экспоненты, логарифма или других базовых функций до определённого предела `N`).
Это классическая задача для отработки аккумуляторов в циклах `for` / `while`.
