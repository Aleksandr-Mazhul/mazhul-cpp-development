# Лабораторная работа 17 (labsV6)

В файлах этой лабораторной работы (`part1.cpp`, `part2.cpp` и `example.cpp`) активно прорабатывается важнейшая и
специфичная тема языков C/C++: **указатели (pointers) и адресная арифметика**.

## Описание

Вместо классической итерации по массиву с квадратными скобочками `array[i]`, здесь используется перемещение по памяти (
от `begin` до `end`).

- Вычисляется скалярное произведение двух "векторов" (массивов) с помощью логики `(*pA) * (*pB)`.
- Указатели инкрементируются (`p++`), обеспечивая перемещение по адресам на размер типа `float`.
- Демонстрируются функции обхода массива, принимающие на вход начало памяти (`float *begin`) и конец памяти (
  `float *end`) по образцу стандартной библиотеки итераторов (STL).
