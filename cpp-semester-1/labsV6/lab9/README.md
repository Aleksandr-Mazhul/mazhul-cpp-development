# Лабораторная работа 9 (labsV6)

В этой папке лежат программы, решающие геометрические и математические задачи на плоскости.

## Описание

В файлах (например, `part1.cpp`) реализуются функции, проверяющие нахождение точки с заданными координатами `(X, Y)` внутри определенной геометрической фигуры. 
- **`insideZone(double, double)`**: Функция для математической проверки попадания точки внутрь окружности радиуса `R` (по теореме Пифагора).
- Использование логических типов `bool` и форматированного вывода с помощью `std::boolalpha`.
- Изучение базовых математических функций из заголовочного файла `<cmath>` (например, `sqrt`).
