# Лабораторная работа 14 (labsV6)

В папке `lab14` студент закрепляет понимание типов с плавающей точкой (`double`), работы с точностью (`eps`) и форматирования консольного вывода.

## Описание

Практикум посвящён вычислению бесконечных числовых рядов.
- **Интерактивный ввод и валидация**: Бесконечный цикл `while(true)` требует ввести корректные данные (например, `|x| < 1` и положительную, но маленькую точность `eps < 0.05`).
- **Суммирование рядов**: Использование рекуррентных математических формул, когда следующий член ряда вычисляется через значение предыдущего (через умножение `term *=`), что экономит вычислительные ресурсы процессора.
- **Табличный вывод**: С помощью библиотеки `<iomanip>` (манипуляторы `setw`, `left`) и класса `std::string(count, char)` на экран печатается строгая таблица для сравнения кастомной функции против встроенной библиотечной (`std::pow`).
