# Лабораторная работа 8 (labsV6)

Данная папка содержит программы для лабораторной работы по дисциплине "Программирование" — начальное знакомство с
массивами.

## Описание

Задачи (в файлах `part1.cpp`, `part2.cpp`) сфокусированы на базовой манипуляции статическими одномерными массивами в
C++:

- Использование оператора `sizeof(array) / sizeof(type)` для автоматического вычисления длины статического массива.
- Итерация в циклах `for`.
- Расчет средних арифметических (`sum / N`), поиск специфичных элементов, вывод результатов.
