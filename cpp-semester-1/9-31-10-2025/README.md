# Практика 9-31-10-2025 (Ряды Тейлора)

В этой папке реализуются математические расчеты, в частности разложение функций в **Ряд Тейлора** (`taylor.cpp`,
`taylor v2.cpp`).

## Описание

Вычисление функций (например, `sin(x)`, `cos(x)`, `e^x`) через бесконечные суммы (ряды Тейлора/Маклорена) с заданной
точностью `epsilon`. Это невероятно полезное задание на:

- Понимание точности чисел с плавающей точкой (`double`).
- Работу с рекуррентными формулами (как получить следующий член ряда из предыдущего без полного пересчета факториалов и
  степеней).
- Сложные циклы с условием выхода по точности `while (abs(term) > eps)`.
