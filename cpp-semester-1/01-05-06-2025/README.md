В этой директории находятся разделенные по папкам `exercise 1` — `exersises 8-14` самые первые задания по

## Описание

Судя по структуре (одна задача — одна папка), здесь проходило первичное знакомство с IDE CLion, компиляцией программ,
структурой функции `main()` и базовым вводом/выводом. Подобные упражнения помогают привыкнуть к синтаксису C++ и базовым
переменным.
