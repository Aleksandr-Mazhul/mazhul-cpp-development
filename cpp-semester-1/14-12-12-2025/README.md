# Практика 14-12-12-2025 (Изучение char)

Файл **`char.cpp`** посвящен символьному типу данных и C-строкам.

## Описание
До перехода к удобному `std::string` в С++ изучаются C-строки (массивы типа `char[]`), оканчивающиеся нуль-терминатором `\0`. Эта работа включает в себя:
- Понимание кодировки ASCII.
- Итерация по строке (через индексы или арифметику указателей).
- Базовые проверки символов (число, буква, пробел).
