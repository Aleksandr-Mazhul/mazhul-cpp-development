# Практика 03-19-06-2025 (Алгоритмы и циклы)

Набор практических заданий (`ex1.cpp` - `ex23.cpp`, `exk.cpp`, `exnum.cpp`), продолжающих тему управляющих конструкций.

## Описание

В коде реализованы различные числовые алгоритмы (вычисление сумм, работа с цифрами числа, циклы). Названия файлов
`exnum` и `exk` указывают на работу с конкретными условиями по цифрам или индексам. Идеальная тренировка на циклы
`while` и `for`.
