# Практика 02-12-06-2025 (Базовые задачи в C++: if/else)

Данная папка содержит набор из нескольких (`ex1.cpp` - `ex9.cpp`) небольших консольных программ на языке C++, написанных в рамках освоения базового синтаксиса и алгоритмов (линейные и разветвляющиеся алгоритмы, условные операторы).

## Структура файлов

- **`ex1.cpp` - `ex8.cpp`**: Каждый файл представляет собой решение отдельной небольшой задачи. Основная сфокусированность — условный оператор `if/else`, работа с переменными, базовые математические расчеты и ввод данных пользователем `std::cin`.
- **`ex9.cpp`, `ex9my.cpp`, `ex9GPT.cpp`**: В этой группе показана эволюция решения 9-й, вероятно, более сложной задачи. Сначала написан свой первоначальный вариант (`ex9my.cpp`), возможно промежуточно работающая версия (`ex9.cpp`), а также вариант с подсказками или оптимизациями от ИИ (`ex9GPT.cpp`). Такое сравнение отлично помогает понять, как можно улучшить и "причесать" собственный логический алгоритм.

## Особенности 

Эти задания — важная основа для понимания того, как в C++ работают управляющие ветвящиеся структуры. Выполнение подобных упражнений закладывает фундамент для более сложных тем (циклы, массивы, указатели, функции), которые размещены в следующих папках.
