# Практика 12-28-11-2025 (Двумерные массивы / Матрицы)

## Описание

Файл **`matrix.cpp`** — это начало работы с двумерными массивами (матрицами). В таких задачах обычно реализуется:

- Выделение памяти под матрицу (статическое или через динамические массивы `new`).
- Заполнение матрицы (ручное или случайными числами).
- Обход двумерного массива через двойные циклы `for(i...){ for(j...) }`.
- Вывод в виде красиво отформатированной таблицы.

Папка `HW/` содержит домашнюю работу по этой теме.
