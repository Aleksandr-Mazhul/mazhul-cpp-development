# Подготовка к экзамену: Маленькие задачи (Small Tasks)

Эта папка содержит небольшие утилиты и сниппеты кода, которые могут потребоваться на экзамене как часть более крупной логики.

## Задачи в наличии:

- **`deleteDIgInNum2.cpp`**: Алгоритм удаления заданной цифры из числа (например, извлечь все двойки из числа 12342). 

Это отличная тренировка для использования математических операций (остаток от деления `% 10` и целочисленное деление `/ 10`) без конвертации числа в массив символов или `std::string`.
