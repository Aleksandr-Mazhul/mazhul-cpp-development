# Практика 13-05-12-2025 (Сортировки)

В папке `sorting/` находятся алгоритмы сортировки данных.

## Описание

Сортировка массивов — важнейшая алгоритмическая тема. Вероятно, в этой директории разобраны базовые методы с
квадратичной сложностью `O(n^2)`:

- **Bubble Sort** (сортировка пузырьком)
- **Selection Sort** (сортировка выбором)
- **Insertion Sort** (сортировка вставками)

Такие алгоритмы показывают работу с обменом элементов (`std::swap`) и сравнением элементов в массиве.
