# Tic-Tac-Toe Game

Интерактивная консольная игра "Крестики-нолики", реализованная в объектно-ориентированном стиле.

---

## Архитектура классов

### `Board`

Отвечает только за состояние игрового поля и правила:

- хранит поле `3x3`;
- проверяет корректность хода;
- ставит символ в клетку;
- проверяет победу/ничью;
- печатает текущее поле.

### `Player`

Представляет игрока:

- имя;
- символ (`X` или `O`);
- получение хода от пользователя.

### `Game`

Управляет игровым циклом:

- хранит поле (`Board`) и двух игроков (`Player`);
- передает ход текущему игроку;
- завершает игру при победе/ничьей;
- переключает активного игрока.

---

## Типы отношений (для разбора на паре)

- `Game` -> `Board`: **композиция** (игра владеет полем).
- `Game` -> `Player`: **композиция** (игра владеет объектами игроков).
- `Player` -> `Board`: **dependency** (игрок использует поле через методы игры, не хранит его как поле).
