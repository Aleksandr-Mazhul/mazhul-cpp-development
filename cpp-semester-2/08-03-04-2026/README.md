# Sorting Utils, Converters, and Comparers

Этот проект представляет собой эволюцию универсального подхода к операциям над массивами данных с использованием
шаблонов (templates) и функциональных объектов C++.

## Описание

Помимо концепции ключей сортировки (Key Extraction), продемонстрированной в предыдущих работах, здесь представлены *
*универсальные компараторы** для гибкого сравнения пар элементов и **шаблонные функции конвертации** одних массивов
данных в другие. В коде также приводится детальное сравнение производительности и возможностей применения собственной
`bubbleSort` наряду со встроенной `std::sort` из алгоритмов STL.

## Функциональность

### 1. Конвертация массивов (`convert`)

Реализован мощный механизм `convert<TIn, TOut>`, который преобразует элементы массива одного типа в другой с
использованием переданной в качестве лямбда-выражения функции трансформации `Converter`.
Примеры из `main`:

- Преобразование вектора `int` в `std::string`.
- Трансформация чисел в их удвоенные строковые представления с добавлением пробела.
- Формирование двоичной маски (`Array<bool>`) на основе условия "больше нуля".

### 2. Сортировка с помощью `Comparer`

В `sorting_utils.h` добавлена перегрузка `bubbleSort`, которая вместо вычисления ключей принимает `Comparer<T>` (
функцию-предикат, возвращающую `bool`), например:

- Сортировка по модулю: `[](int x, int y) { return std::abs(x) < std::abs(y); }`
- Сортировка по возрастанию или убыванию.
- Сортировка по второму символу строкового представления элемента.

### 3. Интеграция с `std::sort`

В коде наглядно тестируется способность кастомных компараторов ложиться поверх стандартной функции `std::sort`. Вы
можете увидеть, что синтаксис для `bubbleSort` и для `std::sort` практически идентичен — это отличная демонстрация
совместимости вашего кода со стандартной библиотекой C++!

## Структура файлов

- `sorting_utils.h` — содержит общие шаблоны данных (`Array<T>`), алиасы `Comparer` и `Converter`, а также реализации
  шаблонных функций `bubbleSort`, `convert` и `display`.
- `sorting_utils.cpp` — содержит реализации базовых ключей `keyAbs`, `keyLastDigit` и специфических функций-сортировок.
- `main.cpp` — тестовый стенд (playground), в котором демонстрируются различные сценарии преобразований типов (через
  `convert`) и сортировки массивов как с помощью `bubbleSort` (с `FunctionKey` и с `Comparer`), так и с использованием
  высокопроизводительной `std::sort`.
