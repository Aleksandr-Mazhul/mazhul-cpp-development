# Sorting Algorithms and CLI Menu

Этот проект представляет собой реализацию различных алгоритмов сортировки на C++ с демонстрацией работы через консоль. В
коде также заложена логика интерактивного консольного меню.

## Описание

Проект (`sort.cpp`) позволяет протестировать классические алгоритмы сортировки на динамически выделенном массиве.
Изначально массив заполняется числами в порядке убывания, чтобы наглядно продемонстрировать процесс сортировки "от
худшего случая". Важной особенностью проекта является использование **указателей на функции** (определенных через
`using Sorting = ...`), что позволяет передавать алгоритмы сортировки в качестве аргументов в функцию-обработчик
`sortArray`.

## Реализованные алгоритмы сортировки

1. **Quick Sort (Быстрая сортировка)** — рекурсивный алгоритм, использующий подход "разделяй и властвуй" с выбором
   опорного элемента.
2. **Bubble Sort (Сортировка пузырьком)** — попарное сравнение элементов и их перестановка.
3. **Selection Sort (Сортировка выбором)** — поиск минимального элемента и перемещение его в начало отсортированной
   части.
4. **Insertion Sort (Сортировка вставками)** — последовательное добавление элементов в правильное место уже
   отсортированной части массива.

## Особенности реализации

- **Указатели на функции**: Вызов `sortArray(quickSort)` позволяет элегантно абстрагировать логику подготовки (выделение
  памяти, вывод на экран) от самого алгоритма.
- **Интерактивное консольное меню (в разработке)**: В коде реализован функционал отрисовки текстового меню (функции
  `textBar`, `menu`, `choiceMenuOption`) и обработки стрелок клавиатуры. Для обеспечения работы функции `getch()` на
  macOS/Linux используется низкоуровневая библиотека `<termios.h>`, которая перехватывает нажатия клавиш без
  необходимости нажимать Enter.
- На текущий момент в `main()` меню закомментировано, и программа последовательно запрашивает размеры массивов для
  тестирования всех четырех сортировок одна за другой.

## Структура файлов

- `sort.cpp` — единый файл, содержащий определения алгоритмов сортировки, утилиты для работы с массивами (инициализация,
  вывод на экран) и логику для построения консольного UI.
