## 🛠 Технические особенности кода

### 1. Использование битовых масок
В файлах акт��вно используются битовые маски для хранения состояния последовательности:
```cpp
enum class TypeOfSequence {
    None   = 0,
    Equals = 1 << 0,
    Less   = 1 << 1,
    More   = 1 << 2,
};
```
Это позволяет комбинировать флаги (например, последовательность может быть одновременно `Less` и `Equals`, то есть неубывающей).

### 2. Перегрузка операторов
В `ex1.cpp` и `notMyCode.cpp` показано, как правильно перегружать побитовый оператор `|` для `enum class`, чтобы операции над флагами были безопасны с точки зрения типизации.

### 3. C++ 20 Features
В `notMyCode.cpp` применяется `using enum TypeOfSequence;`, что позволяет использовать значения перечисления без постоянного указания области видимости, делая код чище.
