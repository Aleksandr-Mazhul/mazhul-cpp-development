# Filters and Predicates

Этот проект демонстрирует использование функций-предикатов и указателей на функции в C++ для фильтрации массивов чисел.

## Описание

Проект реализует различные способы обхода и фильтрации элементов контейнера (например, `std::vector<int>` или кастомного
`IntegerArray`).
Основная идея — показать разницу между жестко закодированными условиями в циклах и гибким подходом с использованием
предикатов.

## Функциональность

В файлах проекта реализованы следующие функции фильтрации:

- **`filterOnlyPositive`**: выводит только положительные числа.
- **`filterOnlyNegative`**: выводит только отрицательные числа.
- **`filterOnlyEven`**: выводит только четные числа.
- **`filterOnlyOdd`**: выводит только нечетные числа.

Кроме того, реализованы универсальные функции с использованием предикатов:

- **`filterByPredicate` (с указателем на функцию `PredicatePointer`)**: принимает массив и указатель на функцию, которая
  определяет, проходит ли элемент фильтр.
- **`filterByPredicate` (с объектом `Predicate`)**: использует современный подход (например, `std::function` или
  шаблонный параметр `Predicate`) и цикл `for (auto number : numbers)`.

## Структура файлов

- `filters.h` и `filters.cpp` — определения и реализации функций фильтрации, как со встроенной логикой, так и с
  использованием передаваемых предикатов.
- `predicates.h` и `predicates.cpp` — реализация конкретных предикатов (функций, возвращающих `bool`), которые можно
  передавать в `filterByPredicate`.
- `main.cpp` — точка входа, где создаются наборы чисел и демонстрируется работа различных методов фильтрации.

## Особенности реализации

- **Гибкость**: Использование предикатов `PredicatePointer` и `Predicate` позволяет не дублировать код циклов каждый
  раз, когда нужно изменить условие фильтрации. Вы просто передаете нужную логику в функцию.
- **Синтаксис C++11**: Использование диапазонного цикла `for (auto number : numbers)` в более современной перегрузке
  `filterByPredicate`.
