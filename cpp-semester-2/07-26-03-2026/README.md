# Generalized Template Sorting (Универсальная сортировка с ключами)

Этот проект демонстрирует мощь шаблонов (templates) C++ и функционального программирования (лямбда-выражений,
`std::function`, указателей на функции) для создания универсальных алгоритмов.

## Описание

Вместо того чтобы писать десятки различных функций сортировки для каждого отдельного случая (по возрастанию, по
убыванию, по модулю, по сумме цифр и т.д.), в данном проекте реализована **одна универсальная шаблонная
функция `bubbleSort`**. Эта функция принимает "функцию-ключ" (key extractor), которая преобразует элемент массива в
значение, по которому и будет производиться сортировка.

## Функциональность

Проект содержит множество вариантов функций-ключей для чисел:

- **`keyAbs`**: сортировка по модулю числа.
- **`keyDigitCount`**: сортировка по количеству цифр в числе.
- **`keySumOfDigits`**: сортировка по сумме цифр числа.
- **`keyLastDigit`**: сортировка по последней цифре.
- **`keyModulo`**: сортировка по остатку от деления.

### Демонстрация в `main()`

В коде наглядно показано использование:

1. **Указателей на функции**: передача обычной функции `keyAbs` для сортировки массива `int`.
2. **Лямбда-выражений с захватом**: сортировка по остатку от деления на динамически заданное число `l` с использованием
   лямбды `[l](int x) { ... }`.
3. **Работы с разными типами данных**: сортировка массива `double` по первому символу их строкового представления (
   преобразование `double` в `char`).

За вывод массивов на экран отвечает удобная шаблонная функция `display`.

## Особенности реализации

- **Проекции (Key Extraction)**: Универсальная `bubbleSort` работает по принципу кэширования ключей. Перед началом
  циклов сортировки создается динамический массив `keys`, в который один раз вычисляются ключи для всех элементов с
  помощью переданной функции. Сама сортировка сравнивает элементы массива `keys`, и при необходимости меняет местами
  элементы параллельно в `keys` и в исходном массиве `arr`. Это значительно снижает количество вызовов ресурсоемких
  функций (например, подсчет суммы цифр).
- **Сравнение подходов**: В файле для наглядности (в закомментированном виде и внизу файла) оставлены классические, "
  жестко зашитые" функции сортировки (например, `bubbleSortBySumOfDigits`). Их наличие подчеркивает, насколько шаблонный
  подход с использованием `FunctionKey` короче и элегантнее, так как избавляет от дублирования логики двойного цикла
  `for`.

## Структура файлов

- `main.cpp` — единственный файл проекта, содержащий вспомогательные функции, устаревшие варианты сортировок, шаблонную
  универсальную функцию и демонстрационный код в `main()`.
