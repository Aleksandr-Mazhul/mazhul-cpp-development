# Лабораторная работа 7: Алгоритмы STL на минималках (Templates)

В этом проекте представлена кастомная (самописная) реализация основных алгоритмов стандартной библиотеки шаблонов (STL) C++. Лабораторная работа является отличным введением в обобщенное программирование (шаблоны C++) и контейнеры фиксированного размера.

## Описание

Проект (`algorithms.h`) состоит из полностью шаблонных алгоритмов, которые работают с контейнером `std::array<T, N>`. Данные функции повторяют поведение известных алгоритмов `<algorithm>`, таких как `std::transform`, `std::count_if`, `std::find_if` и `std::sort`.

Это отличная практика написания типонезависимого и безопасного по типам кода.

## Реализованные функции

- **`initArray`**: Заполняет контейнер `std::array` случайными числами `minValue` до `maxValue`. Благодаря директиве вывода типов `if constexpr (std::is_integral_v<T>)`, функция умеет автоматически переключаться между генераторами целочисленных (`std::uniform_int_distribution`) и вещественных (`std::uniform_real_distribution`) случайных чисел.
- **`printArray`**: Обобщенный вывод массива любого типа в консоль.
- **`transformArray`**: Применяет унарную функцию (или лямбду) к каждому элементу массива и записывает результат на место исходного элемента.
- **`countIf`**: Возвращает количество элементов в массиве, удовлетворяющих заданному условию (`Predicate`).
- **`findIf`**: Ищет и возвращает первый встретившийся элемент, для которого предикат вернул `true`. Если элемент не найден, возвращает инициализацию по умолчанию для типа данных `T` (например, для int это `0`).
- **`sortArray`**: Классическая шаблонная пузырьковая сортировка с кастомным компаратором порядка.

## Использование в `main.cpp`

В точке входа демонстрируется применение этих функций с лямбда-выражениями для:
- Преобразования массива: взятие модуля, если по модулю число `< 10`.
- Поиск количества элементов меньших `-10`.
- Поиск первого элемента больше `5`.
- Сортировка массива по модулям элементов (с лямбда-выражением `abs(a) < abs(b)`).

## Структура файлов

- `algorithms.h` — полностью Header-Only библиотека с шаблонными функциями.
- `main.cpp` — использование библиотеки и пример вызова алгоритмов.
- `tests.h` / `tests.cpp` — автотесты для шаблонных функций.
