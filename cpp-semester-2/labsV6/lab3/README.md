# Лабораторная работа 3: Работа с матрицами и рядами Тейлора

Этот проект представляет собой набор утилит для работы с двумерными динамическими матрицами на C++, а также
демонстрирует вычисление элементов матрицы с использованием как стандартной библиотеки (`cmath`), так и ручной
реализации разложения в ряд Тейлора.

## Описание

Программа позволяет создавать, заполнять и выводить на экран матрицы заданного размера. Ключевой особенностью является
сравнение двух подходов к вычислению значения каждого элемента матрицы:

1. **`buildMatrixLib`**: Использует стандартную функцию `cos()` из `<cmath>`.
2. **`buildMatrixTaylor`**: Использует собственную реализацию `cosTaylor()`, которая вычисляет косинус через его
   разложение в ряд Маклорена (частный случай ряда Тейлора).

Проект также включает в себя набор модульных тестов для проверки корректности вычислений.

## Функциональность

- **Управление памятью**:
    - `allocateMatrix()`: Выделяет память для двумерного массива (массива указателей на массивы).
    - `freeMatrix()`: Корректно освобождает всю выделенную под матрицу память.
- **Заполнение матриц**:
    - `buildMatrixLib()`: Заполняет матрицу значениями, вычисленными по формуле `cos(i) / (j + 1)`.
    - `buildMatrixTaylor()`: Заполняет матрицу по той же формуле, но `cos(i)` вычисляется с помощью `cosTaylor()`.
- **Вывод**: `printMatrix()` для наглядного отображения содержимого матрицы в консоли.
- **Тестирование (`runAllTests`)**:
    - `testCalcElementLib()` и `testCalcElementTaylor()`: Проверяют корректность вычисления отдельных элементов.
    - `testLibVsTaylor()`: Сравнивает результаты `cos()` и `cosTaylor()` на предмет близости значений (с использованием
      `almostEqual`).
    - `testMatrixBuild()`: Проверяет, что матрицы корректно создаются и заполняются.

## Структура файлов

- `matrix_utils.h` и `matrix_utils.cpp` — библиотека функций для работы с матрицами.
- `main.cpp` — точка входа, где запрашиваются размеры матриц, демонстрируется их создание и заполнение обоими методами,
  а также запускаются все тесты.
- `README.md` — данное описание.

## Особенности реализации

- **Динамические двумерные массивы**: Корректная работа с указателями на указатели (`double**`) для создания матриц
  произвольного размера в heap.
- **Ряд Тейлора**: Функция `cosTaylor(double x)` реализует вычисление косинуса через сумму ряда, что является хорошим
  упражнением на понимание численных методов и работы с циклами и точностью `double`.
- **Модульные тесты**: Наличие тестов показывает хороший инженерный подход к разработке, позволяя убедиться в
  корректности каждого компонента системы.
