# Лабораторная работа 2: Работа со строками (Wide Characters)

Этот проект представляет собой реализацию набора утилит для анализа и обработки строк в C++ с использованием широких
символов (`wchar_t`), что необходимо для корректной работы с кириллицей и многоязычным текстом.

## Описание

Программа решает несколько специфических задач по анализу пользовательской строки. Все алгоритмы написаны вручную без
использования готовых функций из `<string>`, что учит работать с сырыми массивами символов (C-строками) напрямую и
аккуратно управлять памятью.

Для обеспечения корректности работы с русским языком программа использует `std::locale` и потоки широких символов
`std::wcin`, `std::wcout`.

## Функциональность

Основные реализованные задачи обработки строк (`string_utils.cpp`):

- **`geometricMeanOddLowercase`**: Вычисляет среднее геометрическое кодов строчных нечетных букв (или что-то подобное, в
  зависимости от логики).
- **`lengthFirstWordEndingWithConsonant`**: Находит длину первого слова в переданной строке, которое заканчивается на
  согласную букву.
- **`countOfWordsWithRussianLetter`**: Подсчитывает количество слов, содержащих хотя бы одну русскую букву.

Вспомогательные утилиты:

- Проверки символов: `isLowercase()`, `isUppercase()`, `isEnglish()`, `isRussian()`, `isVowel()`, `isSeparator()`,
  `isOdd()`, `isEven()`.
- Навигация по строке: поиск начала и конца слова (`findStartOfWordFromPos`, `findEndOfWordFromPos`), поиск русской
  буквы (`findFirstRussianFromPos`).
- Набор автотестов `runTests()` для проверки работы главных функций.

## Структура файлов

- `string_utils.h` и `string_utils.cpp` — библиотека вспомогательных функций (предикаты для анализа символов и основные
  алгоритмы обработки слов).
- `main.cpp` — точка входа в программу. Настраивает локаль, принимает на вход произвольную строку через
  `wcin.getline(str, StrLen)` (с ограничением в 100 символов) и выводит результаты всех трёх основных операций. Также
  после основной программы автоматически запускаются модульные тесты (`runTests`).
- `README.md` — данное описание.

## Особенности реализации

- **Поддержка Юникода / Русского языка**: Использование локали (`locale("")`) и стандарта `wchar_t` позволяет легко
  работать с многобайтными кодировками символов в Windows / macOS.
- **Ручной парсинг**: Разбор строки на слова осуществляется путем прямого обхода индексов и написания собственных
  проверок на разделители (пробелы, знаки препинания). Это отлично тренирует понимание работы памяти, массивов и
  указателей.
