# 🧪 Лабораторная работа №1 (Улиты для массивов)

[![C++](https://img.shields.io/badge/C++-17-blue.svg)](https://isocpp.org/)
[![Jupyter Notebook](https://img.shields.io/badge/Jupyter-Notebook-orange.svg)](https://jupyter.org/)

Привет! В данной директории находится реализация первой лабораторной работы. В ней написаны базовые утилиты для работы с массивами `double`, а также подготовлен подробный **Jupyter Notebook** с разбором каждой функции.

---

## 📋 Что реализовано

Основные функции находятся в `array_utils.cpp` и включают в себя:
- 🔄 **Реверс массива** (`reverse`)
- ⬅️ **Циклический сдвиг** подмассива влево (`cyclicLeftShift`)
- 🔍 **Поиск** последнего отрицательного элемента на чётной позиции
- 🧹 **Сортировка выбором** (`selectionSort`)
- 🎲 **Инициализация** массива случайными числами (`initArray`)

---

## 🗂 Структура файлов

| Файл | Описание |
|------|----------|
| 📄 `array_utils.h` | Заголовочный файл с объявлениями функций. |
| 📄 `array_utils.cpp` | Реализации всех алгоритмов. |
| 📄 `main.cpp` | Главный файл с демонстрацией работы утилит. |
| 📄 `tests.cpp` | Простые Unit-тесты для валидации работы (реверс, сдвиг, сортировка). |
| 📓 `lab1_description.ipynb` | **Подробный разбор кода** (алгоритмы, оценка сложности `O(n)`, граничные случаи и риски). |

## 🚀 Как запустить тесты и демонстрацию

Для компиляции и запуска можно использовать стандартный компилятор (например, `g++` или `clang++`):

```bash
# Компиляция (так как в main.cpp сейчас подключаются .cpp файлы напрямую)
g++ main.cpp -o main

# Запуск
./main
```

> **Примечание:** В будущем рекомендуется использовать **CMake** для правильной сборки проекта, избегая `#include "array_utils.cpp"`.
