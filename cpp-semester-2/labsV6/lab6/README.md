# Лабораторная работа 6: Функции высшего порядка (Filter & Transform)

В данном проекте демонстрируется использование концепций функционального программирования в C++ (функций высшего
порядка) для обработки массивов — фильтрация (filter) и преобразование (transform) элементов.

## Описание

Основная идея работы заключается в написании функции-агрегатора `process`, которая принимает сырой C-массив и две
функции (или лямбда-выражения) в качестве аргументов:

1. `filter` — предикат, решающий должен ли элемент попасть в итоговый массив.
2. `transform` — функция-преобразователь, которая изменяет значение элемента перед его сохранением.

Функция сама выделяет память под новый массив отфильтрованных и измененных элементов и возвращает указатель на него, а
также отдает новый размер через аргумент по ссылке.

## Функциональность

- `process`:
  Выполняет одновременную фильтрацию и трансформацию массива за один проход `O(n)`. Эффективно управляет памятью —
  сначала выделяет временный буфер размером с исходный массив, после обработки элементов (для исключения избыточного
  расхода памяти) создает массив точно под нужное количество элементов (`count`), копирует туда данные и удаляет
  временный буфер.
- Поддержка `std::function`:
  Благодаря использованию `std::function<bool(int)>` и `std::function<int(int)>`, в `process` можно передавать как
  обычные указатели на функции, так и замыкания (лямбды) с захватом переменных.

## Особенности реализации

- **Управление ресурсами**: Функция возвращает "сырой" указатель (`int*`), поэтому вызывающая сторона (caller) **обязана
  ** самостоятельно вызвать `delete[]` для предотвращения утечек памяти.
- **Граничные случаи**: В коде предусмотрены проверки на передачу пустого массива (`size == 0`) — в этом случае функция
  безопасно возвращает `nullptr`.
- **Перевыделение памяти**: Техника с `int* result = new int[count];` позволяет максимально точно экономить память в
  куче.

## Структура файлов

- `process.h` / `process.cpp` — реализация функции обработки массивов и утилит вывода.
- `tests.h` / `tests.cpp` — модульные тесты для проверки корректности работы с разными лямбда-выражениями.
- `main.cpp` — точка входа с примером использования и запуском тестов.
