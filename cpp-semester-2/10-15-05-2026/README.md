# Josephus Problem (Задача Иосифа Флавия)

Этот проект представляет собой симуляцию классической математической "Задачи Иосифа Флавия" на языке C++ с
использованием контейнера из стандартной библиотеки `std::vector`.

## Описание

По условию задачи, `n` человек стоят по кругу. Начиная с первого, люди отсчитываются по кругу, и каждый `k`-й человек
выбывает (удаляется из круга). Процесс продолжается до тех пор, пока не останется только один человек. Программа
визуализирует этот процесс, выводя информацию о том, кто был удален на каждом шаге, и состояние круга после каждого
полного прохода.

## Функциональность

- **Инициализация круга (`fillVector`)**: Заполняет вектор номерами идущими подряд от 1 до `n` (в текущем примере
  `n = 10`).
- **Симуляция выбывания**: Используется вложенный цикл. Основной цикл (while) работает до тех пор, пока в векторе не
  останется один единственный элемент (`people.size() > 1`). Внутренний цикл (do-while) контролирует прохождение по
  оставшимся элементам и отсчет каждого `k`-го участника (`k = 3`).
- **Удаление элементов**: При достижении счетчиком значения `k` происходит удаление человека из вектора с помощью метода
  `erase()`, после чего индексы корректируются для имитации "замкнутого круга".
- **Форматированный вывод**: Перегружен оператор потокового вывода `operator<<` для `std::vector<int>`, что позволяет
  печатать список оставшихся в живых людей одной короткой командой (например, `std::cout << people << std::endl;`).

## Структура файлов

- `main.cpp` — единственный файл, содержащий всю логику симуляции, настройку параметров `n` и `k`, а также
  вспомогательные функции для заполнения и вывода.

## Особенности реализации

- **Явная симуляция процесса**: В отличие от математического решения задачи Иосифа Флавия (которое вычисляет ответ по
  рекуррентной формуле за O(n)), данный подход напрямую моделирует происходящее. Удаление из `std::vector` с помощью
  `erase` сдвигает все последующие элементы (операция O(n)), что делает общую сложность O(n^2), однако это идеальный
  способ детально проследить за каждым шагом отбора.
- **Работа с индексами по кругу**: Корректная обработка перехода через конец вектора обратно в начало при помощи
  алгоритмических проверок и пересчета индекса (`index = 0` или `index = (index + 1) % people.size()`).
