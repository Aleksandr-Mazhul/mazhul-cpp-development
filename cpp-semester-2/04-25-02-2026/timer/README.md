# Timer

Этот проект представляет собой простую реализацию класса таймера обратного отсчета (`Timer`) на языке C++.

## Описание

Класс `Timer` предназначен для отсчета заданного количества секунд до нуля. Когда таймер запускается, он последовательно уменьшает значение оставшегося времени, выводя его на экран при каждой итерации, пока время не истечет.

## Функциональность

- **Инициализация**:
    - По умолчанию таймер инициализируется значением 9 секунд (через делегирование конструкторов).
    - Можно задать произвольное начальное время в секундах.
    - При попытке передать отрицательное время выбрасывается исключение `std::invalid_argument`.

- **Установка времени (`set(int seconds)`)**:
    - Позволяет изменить оставшееся количество секунд. Как и в конструкторе, отрицательные значения вызывают исключение.

- **Запуск таймера (`start()`)**:
    - Начинает цикл обратного отсчета. На каждой итерации оставшееся время выводится на экран, после чего вызывается скрытый метод `tick()`, который уменьшает счетчик на 1. Цикл продолжается, пока таймер не истечет (`isExpired()`).

## Структура файлов

- `Timer.h` — содержит объявление класса `Timer`. Разделяет публичный интерфейс (конструкторы, методы `set` и `start`) и внутреннюю реализацию (метод `tick`, переменная состояния, проверки на истечение времени).
- `Timer.cpp` — содержит реализацию методов, объявленных в `Timer.h`.
- `main.cpp` — точка входа в программу. *(Примечание: в текущем виде `Timer start();` объявляет функцию `start`, возвращающую `Timer`, а не создает объект и не запускает его, так что может потребоваться корректировка `main.cpp` для корректной демонстрации)*.

## Особенности реализации

- Использование делегирующего конструктора `Timer::Timer():Timer(9)` для того, чтобы избежать дублирования кода.
- Методы, не изменяющие состояние объекта, помечены как `const` (`isExpired`, `getRemaining`).
- Инкапсуляция: методы `tick`, `isExpired` и `getRemaining` скрыты в `private` секции, оставляя пользователя взаимодействовать только через метод `start`.
