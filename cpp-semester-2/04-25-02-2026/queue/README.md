# Queue (Кольцевая очередь)

Этот проект представляет собой реализацию шаблонного класса статической кольцевой очереди `Queue` на языке C++.

## Описание

Класс `Queue<T, size_t size>` реализует структуру данных «очередь» на основе встроенного массива фиксированного размера. Он использует шаблоны (templates) C++, что позволяет создавать очереди для хранения элементов любого типа данных (`T`), размер которых задается на этапе компиляции параметром `size`.

Реализация работает по принципу кольцевого буфера (Circular Buffer). Это означает, что при достижении конца выделенного массива новые элементы начинают записываться в его начало, перезаписывая самые старые данные.

## Функциональность

- **Шаблонность**: Поддержка любых типов данных и настройка максимального размера очереди через параметры шаблона (например, `Queue<int, 10>`).
- **Добавление элементов (`add(T item)`)**: 
  - Размещает новый элемент в позиции `tail`.
  - Трекает количество добавленных элементов.
  - При превышении максимального размера (`size`) индексы `head` и `tail` сбрасываются через операцию взятия остатка от деления (`%= size`), обеспечивая автоматическую перезапись старых данных без выхода за границы массива.

## Структура файлов

- `main.cpp` — содержит определение шаблонного класса `Queue`, реализацию его методов и точку входа `main()`, в которой создается очередь из 10 элементов типа `int` и выполняется циклическое добавление чисел от 1 до 10 для тестирования логики работы с индексами.

## Особенности реализации

- Использование сдвига индексов `tail` и `head` `( %= size )` гарантирует безопасную работу с массивом фиксированного размера в условиях непрерывного потока данных, что отлично подходит для буферов ввода-вывода или логгеров.
