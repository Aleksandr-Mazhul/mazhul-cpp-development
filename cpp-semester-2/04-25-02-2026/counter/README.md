# Counter

Этот проект представляет собой реализацию класса счетчика (`Counter`) на языке C++.

## Описание

Класс `Counter` предназначен для управления целочисленным значением в заданном диапазоне с возможностью инкремента,
декремента и сброса к начальному значению. Класс обеспечивает проверки на выход за границы диапазона, что предотвращает
установку недопустимых значений.

## Функциональность

- **Инициализация**: При создании объекта `Counter` можно задать текущее значение, минимальное значение, максимальное
  значение и шаг изменения.
- **Инкремент (`increment()`)**: Увеличивает текущее значение на заданный шаг, если это не приведет к выходу за верхнюю
  границу (`max`).
- **Декремент (`decrement()`)**: Уменьшает текущее значение на заданный шаг, если это не приведет к выходу за нижнюю
  границу (`min`).
- **Сброс (`reset()`)**: Возвращает счетчик к установленному начальному значению.
- **Установка начального значения (`setInitial(int)`)**: Изменяет значение, к которому происходит возврат при вызове
  `reset()`.
- **Получение текущего значения (`getCurrent()`)**: Возвращает текущее состояние счетчика.
- **Вывод последовательности (`printSequence()`)**: Печатает последовательность значений от текущего до максимального с
  заданным шагом.

## Структура файлов

- `counter.h` — определение класса `Counter` и его методов (задано полностью в заголовочном файле).
- `counter.cpp` — файл с реализацией. В текущей версии методы реализованы прямо в классе в `counter.h`, поэтому этот
  файл содержит пустой или лишь частично заполненный конструктор.
- `main.cpp` — файл для тестирования логики счетчика.

## Особенности реализации

- Реализован конструктор копирования и конструктор перемещения.
- Перегружены операторы присваивания (копирующий и перемещающий).
- Встроенные проверки корректности инициализирующих данных выкидывают исключение `invalid_argument`, если минимальное
  значение больше максимального, или текущее значение вне диапазона `[min; max]`.
